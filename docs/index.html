Features

Everything you need to manage skills

🔍

Discover

Browse skills from the marketplace and community sources. Search, filter by agent, and preview before installing.

⚡

Install to multiple agents

Install a skill to one agent or all of them at once. Skills Manager handles the path differences automatically.

🧪

Sandbox testing

Try a skill before committing. The built-in LLM sandbox lets you see exactly how it performs.

📦

Lifecycle management

Track versions, check for updates, star favorites, and remove skills you no longer use.

🔄

Real-time sync

FSEvents watches your skill directories. Changes from any source appear instantly.

🔒

Fully local

No backend, no account required. Works offline. Your skills stay on your machine.
